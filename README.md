# beauty
## Setup
> yarn install
## Run
> yarn watch
## Build
> yarn build
## Scripts
### Generate fake products/collections/brands
> yarn db:gen 
### Clean products
> yarn db:clean
### Clean images (products/users)
> yarn img:clean
